# Shree Shantadurga Sangodkarin Sabhagruha A/c Hall Website

A modern, responsive, and elegant website for Shree Shantadurga Sangodkarin Sabhagruha A/c Hall, built to showcase the venue's amenities, galleries, and provide easy means of booking.

## Tech Stack

This project uses modern web development tools for optimal performance and developer experience:
- **React (v19)**: Component-oriented UI library.
- **Vite**: Next Generation Frontend Tooling for lighting-fast cold loading and hot-module replacement.
- **Tailwind CSS (v3)**: A utility-first CSS framework for rapid and customized styling.
- **React Router (v7)**: Declarative routing.

## Features
- **Responsive Layout**: Seamlessly formats content from mobile screens to desktop widescreen monitors.
- **Dynamic Gallery Grid**: A masonry-styled grid dynamically fetching and allocating venue imagery.
- **Booking Submission**: Form inputs linked directly to a Google Apps Script Web App for intuitive Enquiry/Booking routing to the Management.
- **Quick Links**: Integrated WhatsApp and embedded Google Map capabilities for rapid client interaction.

## Getting Started

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. Clone the repository and navigate to the project root:
   ```bash
   cd temple-hall
   ```

2. Install the necessary dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open your browser and navigate to the given `localhost` link (e.g., `http://localhost:5173/`).

### Building for Production
To generate a production-ready codebase, run:
```bash
npm run build
```
This builds your React app in the `dist` directory which can then be deployed to modern hosting platforms (Vercel, Netlify, Github-Pages, etc.).
